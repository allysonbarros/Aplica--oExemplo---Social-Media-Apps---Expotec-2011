Introdução à Linguagem Scala

Scala (Scalable language) é uma linguagem de programação de propósito geral, diga-se multiparadigma, projetada para expressar padrões de programação comuns de uma forma concisa, elegante e type-safe. Ela incorpora recursos de linguagens orientadas a objetos e funcionais. Também é plenamente interoperável com Java. Scala é a sucessora de Funnel, uma linguagem baseada em join calculus.

OrientDB e Java: Embarcando dados em aplicações

OrientDB faz parte da nova geração de banco de dados NoSQL (Not only SQL), que fornece grande flexibilidade no desenvolvimento de aplicações, permitindo o armazenamento de dados em objetos e documentos, além da saída de dados em JSON -- o que o torna pronto para a Web. Seu ponto forte é a fácil implementação: Não há necessidade de criar serviços burocráticos no computador, simplesmente execute o banco de dados diretamente da sua aplicação. Em adição, ainda suporta transações ACID, consultas SQL e índices rápidos.

Play Framework - Desenvolvimento ágil de aplicações web com Java

O Play! é um framework poderoso, rápido, limpo, dinâmico e eficiente para o desenvolvimento web com Java. Seu principal objetivo é facilitar ao máximo o desenvolvimento de aplicações web de pequeno e médio porte utilizando Java através de uma forma simples, rápida e produtiva.

Programação com Robocode

O software é um jogo de programação em linguagem Java que simula um ambiente em que tanques robôs se enfrentam em combates de destruição total. A idéia principal é que os conceitos de programação sejam aprendidos de forma lúdica, estimulando o usuário a aprimorar o comportamento do seu robô na medida em que os conceitos são assimilados.
