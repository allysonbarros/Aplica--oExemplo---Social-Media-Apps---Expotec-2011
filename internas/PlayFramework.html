Play Framework - Desenvolvimento ágil de aplicações web com Java

Descrição: O Play! é um framework poderoso, rápido, limpo, dinâmico e eficiente para o desenvolvimento web com Java. Seu principal objetivo é facilitar ao máximo o desenvolvimento de aplicações web de pequeno e médio porte utilizando Java através de uma forma simples, rápida e produtiva.
Pré-Requisitos: Conhecimentos da orientação à objetos com Java e HTML / CSS básicos.
Data/Horário: de 15/12/2011 10:00:00 a 17/12/2011 12:00:00
Palestrante(s): ALLYSON BRUNO CAMPOS BARROS VILELA / PHILIPPI SEDIR GRILO DE MORAIS
