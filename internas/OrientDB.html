OrientDB e Java: Embarcando dados em aplicações

Descrição: OrientDB faz parte da nova geração de banco de dados NoSQL (Not only SQL), que fornece grande flexibilidade no desenvolvimento de aplicações, permitindo o armazenamento de dados em objetos e documentos, além da saída de dados em JSON -- o que o torna pronto para a Web. Seu ponto forte é a fácil implementação: Não há necessidade de criar serviços burocráticos no computador, simplesmente execute o banco de dados diretamente da sua aplicação. Em adição, ainda suporta transações ACID, consultas SQL e índices rápidos.
Pré-Requisitos: Conhecimento básico em Orientação a Objetos em Java
