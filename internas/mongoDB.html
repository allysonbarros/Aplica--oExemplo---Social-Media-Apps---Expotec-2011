Banco de Dados NoSQL com mongoDB

Descrição: O minicurso será ministrado com 2 h pela manhã e 2 horas a tarde e tem por objetivo falar sobre bancos de dados NoSQL, um novo conceito de persistência de dados. Será usado como exemplo desse tipo de SGBD, o MongoDB que é um banco de dados orientado a documentos, que possui um formato de classes. Esse paradigma utiliza uma tecnologia chamada BSON (Binary Data Script Oriented Notation, Roteiro de Notação Orientado a Dados Binários), que permite ao SGBD construir índices e comparar documentos requisitados tanto em alto nível, de aplicação, quanto no nível mais baixo, internamente ao banco de dado.
Pré-Requisitos: SO Linux, Banco de Dados
Data/Horário: de 16/12/2011 10:00:00 a 16/12/2011 18:00:00
Palestrante(s): DANIELE MONTENEGRO DA SILVA BARROS
