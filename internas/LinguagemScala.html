Introdução à Linguagem Scala

Descrição: Scala (Scalable language) é uma linguagem de programação de propósito geral, diga-se multiparadigma, projetada para expressar padrões de programação comuns de uma forma concisa, elegante e type-safe. Ela incorpora recursos de linguagens orientadas a objetos e funcionais. Também é plenamente interoperável com Java. Scala é a sucessora de Funnel, uma linguagem baseada em join calculus.
Pré-Requisitos: Experiência com alguma linguagem de programação
Data/Horário: de 16/12/2011 08:00:00 a 16/12/2011 16:00:00
