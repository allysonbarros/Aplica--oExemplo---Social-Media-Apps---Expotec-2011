Programação com Robocode

Descrição: Minicurso de programação utilizando o software Robocode. O software é um jogo de programação em linguagem Java que simula um ambiente em que tanques robôs se enfrentam em combates de destruição total. A idéia principal é que os conceitos de programação sejam aprendidos de forma lúdica, estimulando o usuário a aprimorar o comportamento do seu robô na medida em que os conceitos são assimilados.
Pré-Requisitos: conhecimento de algoritmos e alguma linguagem de programação
Data/Horário: de 16/12/2011 16:00:00 a 17/12/2011 18:00:00
Palestrante(s): CLEMILSON DE LIMA MORAIS
